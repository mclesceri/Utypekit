== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `ninja-forms-campaign-monitor` directory to your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit the 'Ninja Forms -> Settings' menu item in your admin sidebar
4. Follow the link to get your API Key and your client ID
5. In the Ninja Forms Emails & Actions tab, you can now add a new action to subscribing a person to your Audience.
