Contributor Index
A contributor index is a list of recipe contributors sorted by last name, along with the page number(s) where their recipes appear. If the last name field is left blank, that contributor will be listed alphabetically by first name. Names with multiple spellings or versions will appear multiple times in the Index. Note: Adding a space before or after a first or last name will create duplicate entries!
