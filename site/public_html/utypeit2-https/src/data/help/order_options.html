These options help you set up your U-Type-It™ Online account. You may edit these options within the U-Type-It™ Online at any time after you have set up your account. If you are unsure about this part of your cookbook, you can skip all these steps and simply move on to the sign-up, or you can contact us with your questions.

I want to add more contributors to my order
This option allows you to add people to this order that can help you organize, edit, or enter recipes. There are different levels of participation, and each level has its own set of permissions. The help (?) buttons on this page of the wizard explain these permissions.

I want to set up my cookbook options now
These options determine the overall cookbook order settings, such as the title of the cookbook, the type of book you want to order, the number of books, and settings for the content you'd like to include in the book.

I want to set up my recipe sections (categories)
This page allows you to set up the categories (recipe sections) that will appear in your book. By default we have already set up seven categories for you. They are; Appetizers & Beverages, Soups & Salads, Vegetables, Main Dishes, Breads & Rolls, Desserts and Miscellaneous. You can edit these sections now, or wait until later.

I want to set up my recipe and style options now
These options determine what your recipes are going to look like, and includes some designer options to help make your book look more professional. These options include the recipe format, how recipes flow through the book, and some other content settings.
