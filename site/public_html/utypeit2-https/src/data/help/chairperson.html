There should only be one chairperson per order. Please use the following process to change chairpersons:

NOTE: Only the current chairperson can follow these steps. If you need help switching chairpersons, please contact Customer Service at 1-800-227-7282.

  1. Go to the "Review Users" tab on the left, click the button "Review Members" and find the user you want to make chairperson.
  2. Click the "Details" link on the right hand side to reveal that user's details.
  3. Change that user's Level to "Chairperson," then save.
  4. Go to the "My Account" tab on the left, click the button "Edit My Info."
  5. Set your level to anything other than Chairperson, then click "Save."
